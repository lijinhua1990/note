 emacs的操作

| c-x c-s                   | 保存                                        |
| c-x c-c                   | 关闭emacs                                   |
| c-x k                     | 关闭buffer                                  |
| c-x 1                     | 单一窗口模式                                |
| c-x 2                     | 分割窗口模式                                |
| c-x c-f                   | 新建                                        |
| m-w                       | 复制                                        |
| c-w                       | 剪切                                        |
| c-y                       | 粘帖                                        |
| m-v                       | 上翻页                                      |
| c-v                       | 下翻页                                      |
| c-l                       | 刷新                                        |
| c-p                       | 光标上移(previous)                          |
| c-b                       | 光标左移(backward)                          |
| c-f                       | 光标右移(forward)                           |
| c-n                       | 光标下移(next)                              |
| m-f                       | 前移一个单词                                |
| m-b                       | 后移一个单词                                |
| c-a                       | 移动到行的开头                              |
| c-e                       | 移动到行尾                                  |
| m-a                       | 移动到句子头                                |
| m-e                       | 移动到句子尾                                |
| m-<                       | 移动到文本头                                |
| m->                       | 移动到文本尾                                |
| c-u num                   | 重复执行命令,e.g:c-u 8 c-f 向前移动８个字符 |
| c-g                       | 取消命令                                    |
| delete                    | 删除光标前面的字符                          |
| c-d                       | 删除光标后面的字符                          |
| m-delete                  | 删除光标前面的单词                          |
| m-d                       | 删除光标后面的单词                          |
| c-k                       | 删除光标处到行尾的内容                      |
| m-k                       | 除去到当前句子的末尾                        |
| c-x u                     | undo                                        |
| c-x c-b                   | 列出缓冲列表                                |
| c-x b                     | 到某个缓冲区                                |
| c-x right                 | 下一个缓冲区                                |
| c-x left                  | 上一个缓冲区                                |
| c-x s                     | 保存缓冲                                    |
| c-x                       | 字符扩展，后跟一个字符                      |
| m-x                       | 名字命令扩展，后跟一个长名字                |
| c-z                       | 挂起,用fg或者%emacs返回                     |
| c-x f                     | 数字  改变边界宽度                          |
| c-s                       | 向前搜索                                    |
| c-r                       | 向后搜索                                    |
| c-x o                     | 切换窗口                                    |
| insert/M-x overwrite-mode | 切换插入/覆盖模式                           |
